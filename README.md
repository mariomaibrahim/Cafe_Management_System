# Cafe Management System

A comprehensive desktop application for managing cafe operations, built with JavaFX and MySQL.

## Overview

The Cafe Management System is a user-friendly desktop application designed to streamline cafe operations. This project provides a complete solution for managing staff accounts, inventory, orders, and customer transactions with an elegant and intuitive interface.
## Technologies Used

- **Java**: Core programming language
- **JavaFX**: UI framework for desktop application
- **MySQL**: Database management system
- **JDBC**: Database connectivity
- **CSS**: Custom styling and theming
- **FontIcon (Ikonli)**: Icon integration
